Obsidian Vault API

API para visualizar e navegar pelo seu vault Obsidian de forma interativa

Endpoints da API

GET Grafo de Conexões

/api/graph

Retorna nós e arestas para visualização do grafo de conexões entre notas

GET Lista de Notas

/api/notes

Lista todas as notas com metadados básicos

GET Conteúdo da Nota

/api/note/{note_name}

Retorna o conteúdo de uma nota específica
